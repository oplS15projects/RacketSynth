Overview

RacketSynth is a module for Racket that creates an easy to use interface for creating sound using the Rsound and Racket GUI libraries. It accomplishes this by wrapping the complicated Rsound function with an easy to use GUI.

Concepts Demonstrated

  • Expressiveness is utilized in creating sounds with different waves forms.
  • We have built an abstraction barrier between the user and the underlying implementation, allowing for ease-of-use by someone unfamiliar with coding.
  • Symbolic data was referenced and used in the respective lists for the style property of GUI elements.

Our Favorite Lines of Code

David's Favorite

(SN) Ask David for his favorite lines

[callback (lambda (text event) (begin (keySound (myGetText keyInput)) (send keyInput set-value "")))]

This is a callback that is placed inside of a text field used to listen for keyboard input. The first part of the callback grabs the current content of the text-field and sends it to the 'keySound' procedure, which uses the input to generate sounds. The next line then sets the content of the text-field to be an empty string, effectively clearing it out. This prevents previous inputs from being perpetuated and keeps each keystroke isolated.

Once DrRacket is opened, click "File" in the upper-right-hand corner of the screen then click on the selection "Package Manager..." from the pull-down menu.

Type in rsound into the textbox labeled "Package Source", then click the button "Update" to the right and below.

Once Rsound is installed, navigate back to the main menu and click the "Run" button.

You can then select notes of different pitches and press the play button. You can also type in a frequency and produce a note at that pitch. If you want to disable the sound, you can hit the stop button. You can also generate sound that follows a wave-like distribution for the sine wave. Once you are finished with the program, you can click the X in the upper-right-hand corner to exit the program.
