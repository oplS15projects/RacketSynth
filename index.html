Overview

RacketSynth is a module for Racket that creates an easy to use interface for creating sound using the Rsound and Racket GUI libraries. It accomplishes this by wrapping the complicated Rsound function with an easy to use GUI.

Below is a screenshot of the GUI interface of Racket as of Sunday, April 26th 2015

Concepts Demonstrated

  • Expressiveness is utilized in creating sounds with different waves forms. We have taken the RSound library procedures and simplified them, making the code more expressive and easier to understand.
  • We have built an abstraction barrier between the user and the underlying implementation, allowing for ease-of-use by someone unfamiliar with coding.
  • Symbolic data was referenced and used in the respective lists for the style property of GUI elements.

Our Favorite Lines of Code

David's Favorite

[callback (lambda (text event) (begin (keySound (myGetText keyInput)) (send keyInput set-value "")))]

This is a callback that is placed inside of a text field used to listen for keyboard input. The first part of the callback grabs the current content of the text-field and sends it to the 'keySound' procedure, which uses the input to generate sounds. The next line then sets the content of the text-field to be an empty string, effectively clearing it out. This prevents previous inputs from being perpetuated and keeps each keystroke isolated.

Steven's Favorite

(if(equal? "Notes" (radio-check freq-box)) (play-Note (radio-check note-box) 1 2) (my-play (string->number (myGetText freq-field)) .5 2)))]))

This is the line of code that determines what kind of sounds is played. If the play note box is checked, then it will play the desired note. If it's not checked, it defaults to playing the frequency that the user inputted into the text box. The parameters .5 and 2 are used as default volume and note durations respectively, though currently notes sustain indefinitely

How to Download and Run

Click here to download the current version of RacketSynth. RacketSynth depends upon racket 6.1.1, which can be downloaded here and on an environment such as DrRacket, which comes with the download.

To unzip RacketSynth after you have downloaded it, first go to its containing folder either by navigating through your file directories you saved it in on a Linux system or by navigating to your "Downloads" folder if you are on a Windows system.

You can open the file RacketSynth.rkt directly if DrRacket is your default application for dealing Racket code by going to the File menu, clicking "Open" then navigating and selecting RacketSynth.rkt .

Once DrRacket is opened, click "File" in the upper-right-hand corner of the screen then click on the selection "Package Manager..." from the pull-down menu.

Type in rsound into the textbox labeled "Package Source", then click the button "Update" to the right and below.

Once Rsound is installed, navigate back to the main menu and click the "Run" button.

You can then select notes of different pitches and press the play button. You can also type in a frequency and produce a note at that pitch. If you want to disable the sound, you can hit the stop button. You can also generate sound that follows a wave-like distribution for the sine wave. Enabling keyboard input allows Racketsynth to detect and interpret keystrokes in order to play sounds. Use the center row of keys, from 'A' to 'K' to play an A major scale. Currently, only one note may be played at a time. A note will sustain until the next note is played. Pressing the space bar will stop the current note. Once you are finished with the program, you can click the X in the upper-right-hand corner to exit the program.
